Window management:
  $mod+Return        Open terminal (Alacritty)
  $mod+q             Kill window
  $mod+f             Fullscreen toggle
  $mod+space         Focus toggle tiling/floating
  $mod+r             Resize mode (enter)
  $mod+Shift+e       Exit i3
  $mod+Shift+c       Reload config
  $mod+Shift+r       Restart i3

Focus movement:
  $mod+h / Left      Focus left
  $mod+j / Down      Focus down
  $mod+k / Up        Focus up
  $mod+l / Right     Focus right

Move windows:
  $mod+Shift+h / Shift+Left     Move left
  $mod+Shift+j / Shift+Down     Move down
  $mod+Shift+k / Shift+Up       Move up
  $mod+Shift+l / Shift+Right    Move right

Resize mode keys:
  $mod+r             Enter resize mode
  Return / Escape    Exit resize mode
  h / Left           Shrink width
  j / Down           Grow height
  k / Up             Shrink height
  l / Right          Grow width

Workspaces:
  $mod+1–$mod+0      Switch workspaces 1–10
  $mod+Shift+1–0     Move container to workspace 1–10

Audio:
  XF86AudioRaiseVolume  Volume up
  XF86AudioLowerVolume  Volume down
  XF86AudioMute         Mute audio
  XF86AudioMicMute      Mute mic

Brightness:
  XF86MonBrightnessDown Brightness down
  XF86MonBrightnessUp   Brightness up

Media controls:
  XF86AudioPlay     Play / Pause
  XF86AudioNext     Next track
  XF86AudioPrev     Previous track

Screenshots:
  $mod+Print         Flameshot (selection)
  Print              Full screenshot

Night light:
  $mod+n             Warm color enable
  $mod+Shift+n       Restore normal color

Launchers:
  $mod+a             Rofi app menu
  $mod+p             Rofi power menu
